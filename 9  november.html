Variables

In JavaScript, variables are used to store and manipulate data. Variables are "containers" for storing information.

Rules for JavaScript variable names:

  • Variable names are case sensitive (y and Y are two different variables)
  • Variable names must begin with a letter or the underscore character

    • var:

    Variables declared with var are function-scoped, meaning they are only scoped within the function where they are declared. If declared outside any function, they become globally scoped. var declarations are hoisted to the top of their scope during the compilation phase.

    let :

    let allows you to declare variables that are block-scoped. This means they are only accessible within the block (enclosed by curly braces) where they are defined.

    const :

    const is used to declare constants. Once a value is assigned to a const variable, it cannot be reassigned.
    Data Types

    Java script provides different data types to hold different types of values.there are two types of data types

  • Primitive data type
  • Non-primitive data type

    • Primitive data type

    Primitive data types in programming are basic data types that represent single values. They are the simplest and most fundamental data types and serve as the building blocks for more complex data structures. These are immutable data types.

    number:

    Numbers - are values that can be processed and calculated. You don't enclose them in quotation marks. The numbers can be either positive or negative.

    string :

    Strings - are a series of letters and numbers enclosed in quotation marks. JavaScript uses the string literally; it doesn't process it. You'll use strings for text you want displayed or values you want passed along.

    boolean :

    Boolean (true/false) - lets you evaluate whether a condition meets or does not meet specified criteria.

    undefined :

    Represents an uninitialized or undefined value.

    null :

    Null - is an empty value. null is not the same as 0 -- 0 is a real, calculable number, whereas null is the absence of any value.

    Non-primitive data type

    Non-primitive data types, also known as reference types, are more complex data structures in programming that can hold multiple values and have methods associated with them. It is a user defined data type

    Object :

    Represents a collection of key-value pairs.

    Array :

    Represents an ordered list of values.

    function :

    Represents a reusable block of code
